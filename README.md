# Catalogo API

1. Clonar el repositorio
2. ``` npm install ```
3. Clone el archivo .env.example y renombrelo a .env
4. Configurar las variables de entorno en el archivo .env
5. Levantar la base de datos
``` docker-compose up -d ```
6. Ejecutar SEED
``` localhost:3001/api/seed ```
7. Correr el proyecto
``` npm run start:dev ```
